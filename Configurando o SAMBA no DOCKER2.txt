- Criando baixando a imagem e criando o container:
  Fonte: https://hub.docker.com/r/dperson/samba

-Exemplo:
$ sudo docker run -it --name samba -e TZ=America/Sao_Paulo -p 139:139 -p 445:445 \
            -v /home/fabricio:/mount \
            -d dperson/samba -p

- Entre no container em execussão:
$ docker exec -it samba /bin/sh

- Edite o arquivo de configuração do samba:
# vi /etc/samba/smb.conf

- Adicione as seguintes linhas no final do arquivo "Pressione a tecla insert":
[Servidor]
path = /mount
available = yes
writable = yes
browseable = yes
public = yes

Salve e saida do arquivo "Pressione as teclas Esc :wq"

- Saia do container:
# exit

- Verificando o help de configuração:
$ docker exec -it samba samba.sh -h

- Adicionando um usuário e senha:
$ docker exec -it samba samba.sh -u "fabricio;teste"

- Reinicie o container:
$ docker restart samba

- Para remover o container em execussão:
$ docker rm -f samba
