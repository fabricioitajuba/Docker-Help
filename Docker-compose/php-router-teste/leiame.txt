-Para construir a imagem do apache :
# docker build -t fabricioitajuba/apache-php-mysqli-pdo-json:latest .

-Para criar os containers:
# docker compose up -d

- Para "derrubar" os containers, apagar os volumes e redes:
# docker compose down -v
