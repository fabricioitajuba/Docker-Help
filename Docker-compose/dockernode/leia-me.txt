Para executar essa aplicação, siga os passos:

- criar um json apartir do npm
$ npm init -y

- instalar o express
$ npm install express

- instalar o nodemon
$ npm install nodemon

alterar o packege.json :

  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },

para:

  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "nodemon index.js"
  },

-criando a imagem:
$ docker build -t fabricio/dockernode .

executando o docker compose:

$ docker-compose up

para executar em background:

$ docker-compose up -d

-para verificar:
localhost:3000
